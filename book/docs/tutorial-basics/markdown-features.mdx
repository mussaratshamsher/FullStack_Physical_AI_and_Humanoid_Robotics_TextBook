## Mathematical Equations

Use LaTeX syntax for mathematical formulations common in robotics:

```md
## Forward Kinematics Equation

The position of the end effector is calculated using:

$$ \mathbf{T} = \prod_{i=1}^{n} \mathbf{A}_i(\theta_i) $$

Where:
- $\mathbf{T}$ is the transformation matrix
- $\mathbf{A}_i(\theta_i)$ is the link transformation matrix for joint $i$
- $\theta_i$ is the angle of joint $i$
```

````md
```python title="src/controllers/pd_controller.py"
import numpy as np

class PDController:
    def __init__(self, kp=10.0, kd=1.0):
        """
        PD controller for robot joint control
        :param kp: Proportional gain
        :param kd: Derivative gain
        """
        self.kp = kp
        self.kd = kd

    def compute_torque(self, desired_pos, actual_pos, desired_vel, actual_vel):
        """Compute control torque using PD law"""
        pos_error = desired_pos - actual_pos
        vel_error = desired_vel - actual_vel

        tau = self.kp * pos_error + self.kd * vel_error
        return tau
````

## Technical Admonitions

Use callouts to highlight important technical information:

```md
:::note Safety Information

When testing robotic control algorithms, always ensure emergency stop procedures are in place and tested before deployment.

:::

:::warning Critical Parameter

Setting the control loop frequency above 1kHz may cause instability in some actuator systems. Test incrementally.

:::

:::caution Risk Assessment

High-torque actuators can cause injury. Implement force limiting and safety boundaries in all control systems.

:::

:::tip Optimization Strategy

For real-time performance, consider using control strategies with computational complexity of O(n) or less for n degrees of freedom.

:::
```

```md
### Denavit-Hartenberg (DH) Parameters
A method to define coordinate frames on robotic linkages. Commonly used for [forward kinematics](./kinematics.md) calculations.

### Torque Control
Method of controlling robotic actuators by specifying the amount of rotational force applied, as opposed to position or velocity control.
```
